# ARCHITECTURE
## Componentes principales
- app/api: endpoints del backend
- app/core: configuración y lógica base
- app/memory: gestión de memoria
- app/services: servicios como LLM, resumen, RAG
- app/tools: lectura de archivos y utilidades
- app/ui: interfaz inicial
- context/: estado del proyecto y handoffs para otras IA
- memory/: personalidad, perfil de usuario, resŻmenes
- data/: bases de datos, vector store y cachés
